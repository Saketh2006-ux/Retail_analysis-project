# Retail Sales Performance Analysis

## Project Overview

This project analyzes retail sales data to uncover business trends, customer purchasing behavior, and profitability patterns.

## Objectives

- Analyze sales performance
- Identify profitable categories
- Visualize sales trends
- Generate business insights

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed

- Data Exploration
- Statistical Analysis
- Monthly Sales Trends
- Category-wise Sales Analysis
- Profit Analysis
- Correlation Analysis

## Key Findings

- Sales vary across categories.
- Profitability differs by product segment.
- Monthly trends reveal seasonal demand patterns.
